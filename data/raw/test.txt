# Python的列表（list）是有序可变的序列，支持索引、切片和列表推导式。
# 字典（dict）是键值对的集合，查找效率为O(1)，适合存储结构化数据。
# 集合（set）是无序不重复的元素集合，常用于去重和集合运算。
# 元组（tuple）是不可变的有序序列，可以作为字典的键使用。

# HTTP状态码分类：2xx成功（200）、3xx重定向（301/302）、4xx客户端错误（404）、5xx服务端错误（500）。
# Cookie和Session都是用于保持用户状态的机制，Cookie存储在客户端，Session存储在服务端。

# 深度前馈网络（DNN）是最基本的深度学习模型，由输入层、隐藏层和输出层组成。
# 卷积神经网络（CNN）通过卷积操作提取局部特征，在图像识别领域表现优异。
# 循环神经网络（RNN）适合处理序列数据，LSTM和GRU是RNN的改进版本。
# Transformer架构基于自注意力机制，是当前NLP领域的主流模型架构。

# 软件设计原则SOLID：单一职责、开闭原则、里氏替换、接口隔离、依赖倒置。
# 设计模式分为创建型（单例、工厂、建造者）、结构型（适配器、装饰器、代理）和行为型（观察者、策略、模板方法）。
